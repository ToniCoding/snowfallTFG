Bienvenido al gestor de archivos de sitio



En este gestor, podrás realizar todo tipo de acciones sobre tu sitio web. Por defecto, verás que hay un único directorio llamado "www". En este directorio tendrás que subir los contenidos de tu sitio web.
¡Ojo! No subas una carpeta que los contenga sino directamente la web, es decir, si tu sitio web tiene tres directorios: HTML, CSS y JS. Sube esos tres directorios, ¡no su carpeta padre!


Para comenzar a visitar tus sitios y gestionarlos, simplemente necesitarás navegar mediante la barra de navegación superior.
Cada apartado hace lo siguiente:

  • Sitios → Desde aquí podrás acceder a la vista de tus sitios web, lo que te permitirá ver datos importantes acerca de ellos.
  • Nuevo sitio → Desde esta ventana podrás abrir un nuevo sitio. Se lleva a cabo todo el proceso de pago y primera subida del sitio.
  • Soporte → En esta ventana podrás acceder al formulario y datos de contacto del soporte. Si tienes algún problema, no dudes en contactarnos de manera inmediata.
